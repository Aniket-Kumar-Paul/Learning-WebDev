#Virtual Hosts :- run multiple websites on one web server
-> search how to set up apache virtual hosts on ubuntu ( in digitalocean )
>cd /etc/apache2/sites-available/ : location of virtual hosts files
>ls
>vim 000-default.conf
->edit the comment(remove #) ServerName with www.domainname.com and DocumentRoot with the path /var/www/<name>
->also add ServerAlias domainname.com

->similarly make different .conf files for different domains
>cp 000-default.conf <new conf file>.conf
>vim <new conf file>.conf

>sudo a2ensite <name.conf> : enables the conf file
>sudo a2dissite <name.conf> : disables the conf file
>systemctl reload apache2 OR service apache2 restart: activate the new config
