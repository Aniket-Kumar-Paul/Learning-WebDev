More on Selectors in CSS

Examples :
  • div p : select all paragraphs inside div
  • div > p : select only paragraphs directly inside div
  • div + p : select paragraphs jiska pichla sibling div hai
  • input[type='text'] : select all inputs whose type is text
  • a[target] : select all anchor texts whose target is set(to anything but set)
  • li:nth-child(3) : select the 3rd child/element inside li
  • li:nth-child(2n+0) : select every 2nd ,i.e child at positions of 2n inside li (n=0,1,2...)[Position starts from 1 not 0]
  • li:nth-child(odd) : select all child at odd positions inside li
